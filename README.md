SillyTavernの拡張機能です

①image-display<br>
背景とは別に画像表示ウインドウを用意して、ウインドウ内の画像をキーワードに応じて自動的に切り替えます。

②chat-window-onoff<br>
チャットウインドウの表示と非表示を切り替えます

③text_linebreaker<br>
行と行の間に自動的に１行分追加して読みやすくします

④text_styling_extension<br>
文字のサイズや色を調整します
