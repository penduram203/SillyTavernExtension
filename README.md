SillyTavernの拡張機能です

# image-display
背景とは別に画像表示ウインドウを用意して、ウインドウ内の画像をキーワードに応じて自動的に切り替えます。

# chat-window-onoff<br>
チャットウインドウの表示と非表示を切り替えれるようになります。

# text_styling_extension<br>
パネルを操作することで文字のサイズや色を調整することが出来ます。
